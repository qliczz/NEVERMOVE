# W.T.H.F. —— Where's The Hell Friend?

> **找找好友究竟在哪**
> *Where the hell is my friend?*

一个面向 **XIVLauncherCN（国服）** 的 Dalamud 纯可视化 QoL 插件。人多、同屏拥挤、好友散落各地时，
帮你用最直观的方式**知道好友究竟在哪**——小地图、大地图、游戏世界、以及原生好友列表，四重定位。

- 纯可视化，无游戏签名依赖，稳定不崩。
- 适配 Dalamud API Level 15 / .NET 10。

---

## 功能一览

### 1. 世界画面高亮（脚下光圈 + 头顶名字）
对在场的好友 / 队友 / 部队成员，在其脚下绘制彩色光圈，头顶显示名字（可选）与距离（可选）。

### 2. 小地图高亮
在小地图上用圆点标出对象位置。缩放档位与锁定（旋转）状态直接读取游戏原生小地图，圆点实时跟随、无需手动校准。

### 3. 大地图高亮
- **相对模式**：以你为视图中心，按实际码数（yalm）投影对象位置。
- **像素级对齐（实验性）**：用 `MapUtil` 投影到大地图矩形，并**跟随地图缩放/平移**——缩放后落点依然正确。
- 显示名字可开关；范围、缩放倍率、方向翻转可在设置里微调。

### 4. 原生描边光效（世界画面）
复用游戏原生「选中目标」的模型轮廓光效（`GameObject.Highlight`），叠加在角色模型上，
与脚下彩色光圈互不冲突。好友 / 队友 / 部队可分别设置描边颜色（游戏内置 7 种预设）与总开关。

### 5. 区域限制
可设置「仅大世界启用」：副本 / 地牢 / 绝境战 / 团队(Raid) / 危命任务(FATE) 等场景**自动不启用**高亮，避免无关干扰。

### 6. 好友列表位置覆盖（原生列表实时覆盖）
打开游戏内 **社交 → 好友**，在每一行右侧**实时覆盖**该好友的位置信息。数据随好友上线 / 移动 / 换图自动刷新。

位置按可获取到的精度分级显示：

| 情况 | 判定 | 显示 | 颜色 |
| --- | --- | --- | --- |
| ① 同图（已加载） | `Location == 当前地图` 且角色同屏渲染 | `同图 X,Z 距离m` | 绿 |
| ① 同图（未加载） | `Location == 当前地图` 但不在渲染范围 | `同图` | 绿 |
| ② 同服异地 | 同服务器、`Location` 可知 | `地图 <地图名>` | 黄 |
| ③ 跨服（同大区） | 不同服务器、同数据中心 | `跨服 <服务器名>` | 橙 |
| ④ 跨大区 | 不同数据中心 | `跨大区 <服务器名>` | 红 |
| 离线 | 不在线 | `离线`（可选，灰色） | 灰 |

> 说明：跨服务器 / 跨大区时，游戏本身不下发精确坐标，插件只能拿到服务器名（与游戏内好友列表一致）。
> 仅「同图且已加载（同屏渲染）」的好友才能取得精确坐标与距离。

## 安装

1. 在 XIVLauncherCN 中启用「自定义插件库」。
2. 添加自定义库地址（本仓库主列表）：
   ```
   https://github.com/qliczz/DalamudPlugins/releases/latest/download/pluginmaster.json
   ```
3. 在 `/xlplugins` 里搜索 **W.T.H.F.** 并安装，重启游戏或重载插件即可。
4. 用命令 `/nevermove` 或 `/nm` 打开设置界面（也可在插件列表齿轮里打开）。

> 插件内部标识（InternalName）仍为 `NEVERMOVE`，升级时会自动覆盖旧版本，无需手动卸载。

---

## 设置项

命令 `/nevermove`（或 `/nm`）打开设置窗口，分页如下：

- **总开关**：游戏画面 / 小地图 / 大地图 高亮；仅大世界启用；好友列表位置覆盖。
- **高亮对象**：好友 / 队友 / 部队 三类分别开关（优先级 好友 > 队友 > 部队）。
- **颜色**：三类对象统一颜色（小地图 / 大地图 / 世界 三处一致）。
- **游戏画面**：光圈半径 / 粗细、名字、距离。
- **小地图**：圆点半径、名字。
- **大地图**：像素级对齐、圆点半径、相对范围、名字、缩放倍率、X/Y 翻转。
- **描边光效**：启用开关 + 好友 / 队友 / 部队 描边颜色（7 种预设）。
- **好友列表**：启用覆盖、同图显示精确坐标、离线标出。

## 已知限制

- 跨服务器 / 跨大区的好友只能显示服务器名，无法显示精确坐标（游戏不下发）。
- 同图但超出渲染距离的好友，只能判定「同图」，无法给出实时坐标。
- 原生描边颜色为游戏内置 7 种预设色，不能选任意 RGB（任意颜色请用世界画面的彩色光圈 / 名字）。

## 开发 / 构建

```bat
build.cmd      :: 设置 DALAMUD_HOME 并 dotnet build -c Release
pack.cmd       :: 产出 dist/NEVERMOVE.zip
```

CI 在打 `v*` 标签时自动构建并发布 `NEVERMOVE.zip` + `pluginmaster.json`（Dalamud 依赖从本仓库 `ci-deps` Release 自托管获取）。
